Currently staffed by ██ researchers and ██ armed personnel.

Emergency Protocols

Code Black: Total breach. Initiate site lockdown. Evacuation prohibited.

Code Orange: Minor containment failure. Deploy MTF units.

Code Green: Drill or false alarm. Resume normal operations.
